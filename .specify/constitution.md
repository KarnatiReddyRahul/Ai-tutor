# Constitution — AI Reverse Tutor

This document governs the AI Reverse Tutor project.

## Principles

1. **Adaptive Learning** — Questions must adapt to student skill levels in real time
2. **Feynman Principle** — Teaching proves understanding; the AI interviews, not just answers
3. **Privacy** — API keys are user-provided and never stored server-side
4. **Multilingual** — All interfaces and interviews must support English, Telugu, and Hindi
5. **Quality** — Every change must pass lint, type-check, and maintain >90% coverage
