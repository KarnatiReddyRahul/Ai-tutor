# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | ✅ Supported       |

## Reporting a Vulnerability

Report vulnerabilities by opening a confidential issue or emailing the maintainers.

Do **not** post security issues publicly.

## Responsible Disclosure

- Do not access or modify user data without permission
- Do not perform attacks that degrade service availability
- Report issues promptly

## What We Expect

- Prompt acknowledgment of your report
- Regular updates on remediation progress
- Credit for verified reports (if desired)
