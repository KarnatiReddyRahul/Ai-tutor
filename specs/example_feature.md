# Feature: Adaptive Interview

## Description

The AI conducts an adaptive interview on a selected topic, adjusting question difficulty based on student responses.

## Acceptance Criteria

- [ ] Student selects a topic from the available list
- [ ] AI asks the first question at a default difficulty
- [ ] Correct answers increase difficulty; incorrect answers decrease it
- [ ] Each answer is evaluated for misconceptions
- [ ] Interview ends after 10 questions or when skill is sufficiently assessed
- [ ] Results are saved to assessment history
